# Raportarea activității științifice – Irina Vilciu

## Comisia 9 – Inginerie electrică

Acest repository este destinat generării unui raport Excel privind
publicațiile și citările Irinei Vilciu, pe baza profilului public
Google Scholar și a criteriilor prevăzute de OMEC nr. 3018/2025.

## Surse utilizate

- Profil Google Scholar:
  https://scholar.google.com/citations?user=sSYoPq0AAAAJ

- [OMEC nr. 3018/2025 – standarde minimale pentru conferirea diplomei de doctor](OMEC_3018-2025.pdf)

## Criterii pentru Comisia 9 – Inginerie electrică

Pentru îndeplinirea standardului minimal sunt necesare:

1. minimum două lucrări publicate;
2. minimum o lucrare indexată Web of Science sau IEEE Xplore;
3. minimum o lucrare indexată într-o bază de date internațională recunoscută;
4. candidatul să fie autor principal la minimum una dintre lucrări;
5. prezentarea unor dovezi verificabile pentru fiecare activitate declarată.

## Rezultatul proiectului

Proiectul va genera un fișier Excel care va conține:

- lista publicațiilor;
- numărul citărilor Google Scholar;
- metadatele bibliografice;
- clasificarea WoS, IEEE Xplore și BDI;
- verificarea calității de autor principal;
- evaluarea îndeplinirii criteriilor minimale.
